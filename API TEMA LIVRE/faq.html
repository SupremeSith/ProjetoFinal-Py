FAQ - Sistemas Financeiros

Quais são os métodos de pagamento aceitos?

Aceitamos pagamentos via cartão de crédito e transferência bancária.

Como posso visualizar meu extrato financeiro?

Você pode acessar seu extrato financeiro através da nossa plataforma online, na seção de contas.

Existe algum custo associado ao uso do sistema?

Oferecemos uma versão gratuita com funcionalidades básicas. Para recursos avançados, consulte nossos planos pagos, detalhes disponíveis em nossa página de preços.

Posso importar dados de outros aplicativos financeiros para este sistema?

Sim, você pode importar dados de outros aplicativos financeiros.

Como posso editar ou excluir uma transação já registrada?

Acesse a seção de transações e selecione a transação desejada. Lá, você encontrará opções para editar detalhes ou excluí-la.
